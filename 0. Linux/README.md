### Структура Каталогов

|Каталог|Назначение|Примеры статей / Темы|
|---|---|---|
|`01_Foundations/`|Базовые концепции и основы работы Linux-системы.|Устройство файловой системы (FHS), процесс загрузки, основы прав доступа.|
|`02_CLI_and_Shell/`|Все, что связано с командной строкой, оболочками (shell) и скриптингом.|Синтаксис Bash и Zsh, утилиты `grep`, `awk`, `sed`, написание скриптов.|
|`03_System_Administration/`|Заметки по управлению системой, сервисами, сетями и оборудованием.|Управление сервисами systemd, `nmcli`, `iproute2`, настройка фаерволов, **OOM Killer**, работа с ядром.|
|`04_Software_Management/`|Установка, обновление и управление программным обеспечением.|Работа с пакетными менеджерами (APT, DNF, Pacman), сборка программ из исходников.|
|`05_Environments/`|Статьи, посвященные рабочим окружениям (DE) и оконным менеджерам (WM).|Настройка GNOME, KDE, i3, Hyprland, Wayland и Xorg.|
|`06_Tools_and_Utilities/`|Общие инструменты и приложения, используемые в экосистеме Linux.|Руководства по Git, Docker, Vagrant, SSH, Rsync и другим полезным утилитам.|
