## Что такое переменные окружения?

**Переменная окружения** — это значение, которое доступно для программ в твоей системе. Она может влиять на то, **как работает софт**, не меняя его код.

Примеры:

- `PATH` — где искать исполняемые файлы
- `HOME` — путь к твоей домашней директории
- `LIBVIRT_DEFAULT_URI` — куда подключаться по умолчанию
- `TOFU_EDITOR` — какой редактор использовать в OpenTofu

 `export` — как работает?

Команда `export` делает переменную **доступной для всех процессов**, запущенных из текущей оболочки (и её потомков).
### Пример: 
```
export NAME="Артём"
echo "Привет, $NAME"
```
Выведет: `Привет, Артём`
> `$NAME` — так обращаются к значению переменной
### Как задать переменную:
```
export ИМЯ="значение"
```
Примеры:
```
export LIBVIRT_DEFAULT_URI="qemu:///system"
export TOFU_EDITOR="nano"
export EDITOR="vim"
export AWS_REGION="us-east-1"
```
### Как проверить, установлена ли переменная?
```
echo $ИМЯ
```
Пример: 
```
echo $LIBVIRT_DEFAULT_URI
```
Должно вывести: `qemu:///system`
### Как удалить переменную?
```
unset ИМЯ
```
Пример:
```
unset LIBVIRT_DEFAULT_URI
```
## Где хранить export, чтобы он работал всегда?

Если ты пишешь `export` в терминале — он работает **только в этой сессии**. Закрыл терминал — переменная пропала.

Чтобы она была **всегда**, добавь её в файл конфигурации оболочки.

### Узнай, какая у тебя оболочка:
```
echo $SHELL
```

В зависимости от оболочки, открой source файл (.bashrc .zshrc) и добавь в конец:
```
export LIBVIRT_DEFAULT_URI="qemu:///system"
export EDITOR="nano"
```
и перечитай:
```
source ~/.zshrc
```
