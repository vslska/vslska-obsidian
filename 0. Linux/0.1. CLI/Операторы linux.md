В этой заметке собраны основные операторы, используемые в командной оболочке Linux (Bash), сгруппированные по их назначению.

### 1. Операторы связывания и управления потоком выполнения
Используются для объединения нескольких команд в одну строку и управления их последовательностью выполнения в зависимости от статуса завершения (`0` — успех, любое другое число — ошибка).

| Оператор | Название            | Описание                                                                                             |                             |
| -------- | ------------------- | ---------------------------------------------------------------------------------------------------- | --------------------------- |
| ;        | Точка с запятой     | Выполняет следующую команду безусловно, независимо от результата предыдущей.                         | `cmd1 ; cmd2`               |
| &        | Фоновый режим       | Запускает команду в фоновом режиме, не блокируя терминал.                                            | `sleep 600 &`               |
| &&       | Логическое И (AND)  | Выполняет следующую команду, только если предыдущая **успешна**.                                     | `make && sudo make install` |
| \| \|    | Логическое ИЛИ (OR) | Выполняет следующую команду, только если предыдущая **провалилась** (вернёт ошибку).                 | `nano text1 \|\| vi text1`  |
| \|       | Конвейер (Pipe)     | Перенаправляет стандартный вывод (stdout) первой команды на стандартный ввод (stdin) второй команды. | `history \| grep "man"`     |

### 2. Операторы перенаправления ввода/вывода
Управляют потоками данных между командами, файлами и терминалом.

| Оператор | Название               | Описание                                                        | Пример                      |
| -------- | ---------------------- | --------------------------------------------------------------- | --------------------------- |
| `>`      | Перезапись вывода      | Перенаправляет stdout в файл, **перезаписывая** его содержимое. | `echo "hi" > file.txt`      |
| `>>`     | Добавление вывода      | Перенаправляет stdout в файл, **добавляя** в конец.             | `echo "hi" >> file.txt`     |
| `<`      | Перенаправление ввода  | Использует содержимое файла как stdin для команды.              | `wc -l < file.txt`          |
| `2>`     | Перенаправление ошибок | Перенаправляет поток ошибок (stderr) в файл.                    | `./script.sh 2> errors.log` |
| `&>`     | Объединение потоков    | Перенаправляет и stdout, и stderr в один файл.                  | `cmd &> all_output.log`     |
### 3. Специальные операторы оболочки
Эти операторы используются для сокращения синтаксиса, создания шаблонов и замещения процессов.

| Оператор    | Название            | Описание                                                                           | Пример                       |
| ----------- | ------------------- | ---------------------------------------------------------------------------------- | ---------------------------- |
| `{}`        | Фигурные скобки     | Генерирует набор строк (Brace Expansion).                                          | `touch file_{1..3}.txt`      |
| `$(...)`    | Замещение команд    | Выполняет команду внутри скобок и заменяет её **результатом** (рекомендуется).     | `echo $(date +%F)`           |
| `` `...` `` | Обратные кавычки    | То же, что и `$(...)`, но старый синтаксис (менее гибкий).                         | `echo` date                  |
| `<()`       | Замещение процессов | Предоставляет вывод команды в виде временного файла для другой команды.            | `diff <(ls dir1) <(ls dir2)` |
| `!`         | Отрицание           | Инвертирует логический статус команды или используется в поиске по истории (`!$`). | `if [ ! -f file.txt ]`       |

### 4. Арифметические операторы
Используются внутри конструкции `(( ... ))` для математических вычислений в Bash-скриптах.

| Оператор   | Описание                    | Пример          |
| ---------- | --------------------------- | --------------- |
| `+`, `-`   | Сложение, вычитание         | `((a = 1 + 1))` |
| `*`, `/`   | Умножение, деление          | `((b = a * 3))` |
| `%`        | Остаток от деления (модуль) | `((c = 5 % 2))` |
| `++`, `--` | Инкремент, декремент        | `((a++))`       |
