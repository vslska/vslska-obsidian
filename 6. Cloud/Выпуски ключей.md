### 1. YC
```bash
# Создать сервисный аккаунт
yc iam service-account create --name terraform-sa

# Назначить роль (например, admin на каталог)
yc resource-manager folder add-access-binding <FOLDER_ID> \
  --role admin \
  --subject serviceAccount:<SA_ID>

# Создать статический ключ в формате Terraform
yc iam key create \
  --service-account-name terraform-sa \
  --output sa-key.json \
  --format json
```
В Terraform 
```
provider "yandex" {
  service_account_key_file = "sa-key.json"
  folder_id                = "..."
}
```
