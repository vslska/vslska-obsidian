
 #### Запуск плейбука
 Можно запускать динамически, указав группу, даже если в  плейбуке указан hosts
 пример:
```
 ansible-playbook --limit
```
#### Контакенация строк  ~ ' ' ~

Например мне необходимо объединить значения из setup

```
- name: Сбор данных с сервера
  debug:
    msg: "{{ {
       'ОС' : ansible_distribution ~ ' ' ~ ansible_distribution_version
  } }}"
```

В данном случае мы объединили  название ОС и версию ОС в одну строку.
#### Как сделать  Вывод построчно:

добавляем к переменной stdout (вывод, без дополнительной информации дебага)
_lines выводит построчно

```
- name: сбор данных о пользователях
  shell: who | sort -u
  register: active_user
  args:
     executable: /bin/bash

- name : Отобразить
  debug:
-   msg: "{{ active_users.stdout_lines}}"
```
