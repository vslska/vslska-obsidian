#### Удаление созданных ресурсов

> [!NOTE]  Важно
> С помощью Terraform остановить ресурсы нельзя. Поэтому для экономии денег ресурсы лучше **удалить**

Удалить ресурсы можно  с помощью команды
```hcl
$ terraform destroy
```

В результате её выполнения появится план изменений и будет запрошено подтверждение на его применение.

#### На этом этапе происходит:
- Удаление ресурсов, указанных в конфигурации
- Обновление state
