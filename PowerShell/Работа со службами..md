Эти командлеты позволяют вам получать информацию, а также запускать, останавливать, перезапускать и настраивать службы.

В Windows PowerShell управление службами осуществляется с помощью **встроенных командлетов** семейства `Service`. Для выполнения большинства операций, связанных с изменением состояния или параметров служб (запуск, остановка, изменение типа запуска), требуется запуск PowerShell **от имени администратора**.

## 1. Основные Командлеты

Все основные операции выполняются с помощью следующих командлетов:

|Командлет|Описание|
|---|---|
|`Get-Service`|Получает список служб или информацию о конкретной службе.|
|`Start-Service`|Запускает остановленную службу.|
|`Stop-Service`|Останавливает работающую службу.|
|`Restart-Service`|Перезапускает службу (останавливает и запускает снова).|
|`Set-Service`|Изменяет параметры службы (тип запуска: автоматический, ручной, отключен).|
|`New-Service`|Создает новую службу (требует исполняемого файла).|

## 2. Примеры Использования (Сниппеты Кода)

Ниже приведены практические примеры использования командлетов.

### Получение информации

Получение списка всех служб:
```powershell
Get-Service
```

Получение информации о конкретной службе (например, служба обновлений Windows с именем `wuauserv`):
```powershell
Get-Service -Name wuauserv
```

Получение списка только **работающих** служб с использованием фильтрации через `Where-Object` (псевдоним `where`):
```powershell
Get-Service | Where-Object {$_.Status -eq "Running"}
```

### Управление состоянием служб

_Для выполнения этих команд требуется запуск PowerShell **от имени администратора**._

Остановка службы (например, службы печати `spooler`):
```powershell
Stop-Service -Name spooler
```

Запуск службы:
```powershell
Start-Service -Name spooler
```


Перезапуск службы (остановка и последующий запуск):
```powershell
Restart-Service -Name spooler
```
### Изменение параметров запуска

Изменение типа запуска службы на **Автоматический** (`Automatic`):
```powershell
Set-Service -Name wuauserv -StartupType Automatic
```

Изменение типа запуска на **Отключено** (`Disabled`):
```powershell
Set-Service -Name wuauserv -StartupType Disabled
```

Другие возможные значения для параметра `-StartupType`: `Manual` (вручную).
